# TiPowerUp Template

TastyIgniter extension template for TiPowerUp

## Installation

```bash
composer require tipowerup/ti-ext-template
```

Then install in TastyIgniter:

```bash
php artisan igniter:extension-install tipowerup.template
```

## Requirements

- TastyIgniter v4.0+
- PHP 8.2+
